<h2>How the Minting Bot Works: Step-by-Step</h2>

<ol style="max-width: 800px; margin: 2em auto; font-size: 1.1em; line-height: 1.8; color: var(--text-secondary);">
  <li><strong>Starts with Mining Revenue:</strong> Uses earnings from miners to fund the process.</li>
  <li><strong>Checks for CYPU Availability:</strong> Looks where to obtain CYPU – is minting still unfinished?</li>
  <li><strong>If Unfinished:</strong> Mints 1 new CYPU directly.</li>
  <li><strong>If Finished:</strong> Places an order to buy 1 CYPU for exactly 1 KAS.</li>
  <li><strong>Storage:</strong> Transfers minted/bought CYPU to a secure wallet for ecosystem use.</li>
</ol>

<p style="text-align: center; margin: 2em 0; font-size: 1.1em; color: var(--text-secondary);">
  This simple cycle ensures fair, backed minting without excess creation, stopping once the cap is hit.
</p>
